# **dna_analyzer_tools**

este é um projeto para **aprendizado de programaçao em python aplicado a biologia**
a ideia é criar um **analizador de DNA**, a primeira versão recebe um Input de uma 
seguência de DNA só aceitando A,T,C,G, conta quantas bases tem de cada uma, e 
possui uma func que transcreve o DNA para RNA substituindo T por U,

**pretendo futuramente adcionar:**

- uma func que calcula a **"rigidez"** da fita de DNA baseado na quantidade de 
ligações de CG

- uma func que calcula o tamnho aproximado em nanometros ou em micrometros das
fitas de DNA para obter dados interessantes para fazer o **"DNA Origami"** sabendo
o tamnho do **Scaffolder** e o tamanho necessário dos **Staples**

- **e futuras ideias**
